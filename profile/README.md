> **"Mods that solve the fiddly bits so you can get back to building weird stuff in space."**

Home for Chris Carpenter's ([Godimas101](https://github.com/Godimas101)) published game mods — Space Engineers and MechWarrior 5, both distributed on Steam Workshop. Every mod here has its own repo, its own project board, and its own bug-report link.

## 🎮 Space Engineers mods

| Mod | What it does |
|---|---|
| [**InfoLCD — Apex Update**](https://github.com/gitpush-mod/se-infolcd-apex-update) | Flagship. Adds real-time info displays to Apex LCD blocks. Client-side only. 1,335+ subscribers. |
| [**InfoLCD — Apex Advanced**](https://github.com/gitpush-mod/se-infolcd-apex-advanced) | Sibling variant of InfoLCD with expanded readouts. |
| [**Not Just For Looks**](https://github.com/gitpush-mod/se-not-just-for-looks) | Upgrades Keen DLC reskins with real gameplay functionality. |
| [**NJFL — Additions**](https://github.com/gitpush-mod/se-not-just-for-looks-additions) | Extension pack for Not Just For Looks. |
| [**NJFL — Without Weapons**](https://github.com/gitpush-mod/se-not-just-for-looks-without-weapons) | NJFL variant with weapon blocks removed. |
| [**Tiered Build and Repair**](https://github.com/gitpush-mod/se-tiered-build-and-repair) | Tiered progression for Build-and-Repair mechanics. |
| [**SG Core Mods**](https://github.com/gitpush-mod/se-sg-core-mods) | Bundle repo: 5 Sturmgrenadier Core mods maintained together. |
| [**Mod Adjuster Mods**](https://github.com/gitpush-mod/se-mod-adjuster-mods) | Bundle repo: 10 balance-adjustment mods for SG servers. |

## 🐛 Report a bug / request a feature

Every mod repo has its own **Issues** tab with pre-filled bug-report and feature-request templates. Click any mod above → **Issues** → **New issue**.

If you're not sure which mod is involved, open the issue on the mod repo you were using — Chris will re-route it if needed.

## 🧡 Support

These mods are free and always will be. If they save you time or make your builds better, consider supporting on Patreon — it's how the tools and mods keep coming.
