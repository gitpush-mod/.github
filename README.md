# gitpush-mod / .github

Special repo for the [`gitpush-mod`](https://github.com/gitpush-mod) org. Two jobs:

1. **`profile/README.md`** — renders as the org's public landing page at [github.com/gitpush-mod](https://github.com/gitpush-mod).
2. **`.github/`** — default community health files that apply to any repo in the org that doesn't have its own:
   - `ISSUE_TEMPLATE/bug_report.md` + `feature_request.md` + `config.yml`
   - `CONTRIBUTING.md`
   - `CODE_OF_CONDUCT.md`

Individual mod repos can override these with their own copies in their own `.github/` folder if they need mod-specific templates.

## Editing

Just push to `main`. GitHub picks up changes to `profile/README.md` on the org landing page immediately (may need a hard refresh).

## Where the actual mods live

Everything else. See the org page or the [`REPO_FAMILY.md`](https://github.com/Godimas101/personal-projects/blob/main/docs/style-guides/REPO_FAMILY.md) reference in `Godimas101/personal-projects` for the full list.
